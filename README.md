# 🎓 CampusOne – Smart Student Life Management Platform

> One app for everything a student needs.

CampusOne is a full-stack student productivity and career management platform designed to help students manage academics, placements, coding progress, and daily activities in one place.

---

## 🚀 Features

### 📊 Smart Dashboard
- Attendance overview
- Upcoming deadlines
- GPA overview
- Recent notifications

### 📚 Attendance Management
- Subject-wise attendance tracking
- Low attendance alerts
- Attendance prediction

### 📝 Assignment Manager
- Create and manage assignments
- Due dates and priorities
- Assignment status tracking

### 📂 Notes & Resources
- Upload and manage notes
- Subject-wise organization
- Search functionality

### 📅 Smart Timetable
- Weekly class schedule
- Exam timetable
- Event reminders

### 📈 GPA & CGPA Calculator
- Semester GPA calculation
- CGPA prediction
- Performance graphs

### 🎯 Placement Tracker
- Company applications
- Interview status
- Offer tracking

### 🤖 AI Features (Upcoming)
- AI Study Planner
- Resume Feedback
- Interview Question Generator
- Skill Recommendations

### 🏆 Future Features
- Coding Tracker
- Community Forum
- Gamification System
- Mobile App Version

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/mail2anjaligupta2005-ctrl/CampusOne.git
```

### Backend Setup

```bash
cd backend
npm install
node server.js
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
